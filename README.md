## Requerimientos
<br>
[_>]  Generar una interfaz gráfica de escritorio, que soporte los elementos de un árbol binario de búsqueda y pinte el recorrido tanto en preorden, inorden y postorde.

</details>

| Funciones | Y/N |

| Agregar nodos | ✅ |
| Eliminar nodos | ✅ |
| Recorrer PreOrden | ✅ |
| Recorrer PostOrden |  ✅ |
| Recorrer EnOrden|  ✅ |
| Arbol Simple | ✅ |
| Arvol AVL | ✅|

| Mas funcionalidades Proximamente | ♻️ |
